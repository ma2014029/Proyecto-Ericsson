Procesos

Son mecanismos de comportamiento que diseñan las personas para mejorar la productividad de algo, para establecer un orden o eliminar algún tipo de problema. Es una secuencia de pasos de forma lógica.

Muchos de estos procesos son fundamentales para el mantenimiento del sistema operativo, circunstancia que hace eliminar alguno pueda tener consecuencias negativas.

SDLC (Software Development Life Cycle)

Es un proceso utilizado por la industria del software para diseñar, desarrollar y probar softwares de alta calidad.

Software Testing Life Cycle

Se refiere a un proceso de prueba que tiene pasos específicos que deben ejecutarse en una secuencia definida para garantizar que se cumplan los objetivos de calidad.

Defect Management

Es el seguimiento de defectos es el proceso de rastrear los defectos registrados en un producto desde el principio hasta el cierre, y crear nuevas versiones del producto que solucionen los defectos.
