En el proceso de desarrollo de software es una estructura aplicada al desarrollo de un producto de software Hay varios modelos a seguir para el establecimiento de un proceso para el desarrollo de software, cada uno de los cuales describe un enfoque diferente para diferentes actividades que tienen lugar durante el proceso. Hay varios procesos de desarrollo de software especificos que se ajustan a un modelo de ciclo de vida. Las actividades dependeran del modelo que se elija.
