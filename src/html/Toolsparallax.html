¿Que es?

¡Estamos en una era de automatización en todas partes! estamos entrando en una generación en la que habrá aplicaciones desarrolladas específicamente para que el trabajo se realice de manera fácil y eficiente. Para eso existen las herramientas para el testeo de software son herramientas hechas para, como su nombre lo indica el testeo de software automatizado, y tambien las herramientas de desarrollo de software, en si es un programa informático que usa un programa informático para crear, depurarlo, gestionar o mantener un programa.
En las herramientas para el testep estan: SoapUi, Postman, Jmeter.
En las herramientas para el desarrollo estan: make, Jtest, Javadoc
Algunos ejemplos son:
SoapUi:Es una herramienta desarrollada en java para la realizacion de pruebas a aplicaciones con arquitectura orientada a servicios SOA y transferencias de estado REST.

Postman: Es una herramienta cuando intentas diseccionar API RESTful, hechas por otros o proba las que tu mismo has hecho. Ofrece una interfaz de usuario elegante con la que hacer solicitudes HTML, sin la molestia de escribir un monton de codigo solo para propar la funcionabilidad en una API. Con Postman tal prueba es mucho mas simplificada.

Selenium: Es un entorno de pruebas de software para aplicaciones basadas en la web. Selenium provee una herramienta de grabar/reprudicir para crear pruebas sin usar un leguaje de scripting para pruebas
