Software Development Life Cyclez

Es un marco que define las tareas realizadas en cada paso del proceso de desarrollo de software. SDLC es una estructura seguida por un equipo de desarrollo dentro de la organización de software. Consiste en un plan detallado que describe cómo desarrollar, mantener y reemplazar software específico. El ciclo de vida define una metodología para mejorar la calidad del software y el proceso de desarrollo general.
